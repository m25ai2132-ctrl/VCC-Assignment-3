## Assignment Overview
This assignment demonstrates setting up a **local Ubuntu VM** and integrating it with **Google Cloud Platform (GCP)** to implement **auto-scaling based on resource usage**. The key tasks include:

- Configuring a **Service Account** and downloading the JSON key for authentication  
- Monitoring **CPU and memory usage** on the local VM  
- Writing a **Bash script** to automatically trigger a GCP VM when resource usage exceeds 75%  
- Deploying a **sample application** on the newly created cloud VM to verify auto-scaling  
- Ensuring **security** and proper access control through IAM roles

This setup shows how local and cloud environments can work together to handle workload spikes efficiently.
